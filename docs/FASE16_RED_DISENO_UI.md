# Fase 16 — Rediseño visual estilo RL Analyser

Objetivo: acercar la UI a un cockpit competitivo como las referencias del PDF, sin copiar pixel a pixel.

## Cambios principales

- Shell de escritorio más compacto.
- Sidebar técnica con iconos, navegación densa y estado local.
- Top bar más cercano a herramienta de análisis.
- Dashboard tipo cockpit:
  - rank/MMR card lateral;
  - curva MMR grande;
  - stack de partidas recientes;
  - focus card;
  - heatmap;
  - radar;
  - mapa táctico.
- Sistema visual nuevo:
  - analyzer-card;
  - analyzer-pill;
  - analyzer-button;
  - rank themes dinámicos.
- Game Analysis con mapa táctico tipo annotated replay 2D.
- Heat zones visuales en análisis individual.

## Inspiración visual aplicada

- Página 1: cockpit con curva MMR, heatmap y partidas recientes.
- Página 3/7/11: pantallas por pilares con tendencia y campo táctico.
- Página 8/9/10: game analysis con scoreboard, timeline y visualización espacial.

## No incluido todavía

- Reproductor real de replay con posiciones frame-by-frame.
- 3D renderer.
- OCR local MMR.
- Generador .Tem.

Estas quedan para fases posteriores.
